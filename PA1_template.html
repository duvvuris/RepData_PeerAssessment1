  • Devise a strategy for filling in all of the missing values in the dataset. The strategy does not need to be sophisticated. For example, you could use the mean/median for that day, or the mean for that 5-minute interval, etc. I am using the average no of steps as computed above to replace
totalrows <- nrow(Rawactivitydata)
modactivitydata <- Rawactivitydata
for (i in 1:totalrows) {
    if (is.na(Rawactivitydata$steps[i])) {
        modactivitydata$steps[i] <- activitybyint[activitybyint$interval == 
            Rawactivitydata$interval[i], ]$steps
    }
}
